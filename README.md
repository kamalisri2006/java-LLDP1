# Movie Ticket Booking System

A Java Console-Based **Movie Ticket Booking System** developed using **Object-Oriented Programming (OOP)** concepts. This application enables users to efficiently manage movie details and perform ticket booking operations through a simple menu-driven interface.

**Description**

The Movie Ticket Booking System is designed to simplify the process of managing movies and booking tickets. It allows users to add and view movie details, search for movies, book tickets, cancel movies, and check seat availability. The project demonstrates the implementation of core Java programming concepts and object-oriented principles in a real-world console application.

**Features**

- Add new movies
- View all available movies
- Search movies by ID
- Book movie tickets
- Cancel movies
- Display available seats

**Technologies Used**

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework
- ArrayList
- Scanner Class

**How to Run**

1. Clone the repository.
2. Open the project in your preferred Java IDE.
3. Compile all Java source files.
4. Run the `Main.java` file.

**OOP Concepts Used**

- Classes and Objects
- Encapsulation
- Constructors
- Method Overloading
- Collection Framework
